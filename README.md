# Express で GraphQL 入門

勉強会発表用サンプルプログラム

## サンプルプログラムを動かしてみる
- Hello World を返すだけのGraphQLサーバ
  `node hello-world-server.js`

- コース情報を取得するGraphQLサーバ（参考サイトのやつ）
  `node course-server.js`
  
- コース情報を取得するGraphQLサーバ 簡略版
  `node simple-course-server.js`
